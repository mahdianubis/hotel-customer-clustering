# :hotel: Hotel Customer Segmentation (Clustering)

This project applies Unsupervised Machine Learning to segment hotel guests based on their booking behavior and demographics. By identifying distinct customer groups, hotels can optimize pricing, marketing strategies, and cancellation policies.

---
## :hammer_and_wrench: Tech Stack
- Language: Python
- Libraries: Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
